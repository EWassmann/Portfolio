Sterile Processing Automation

At Surgical Vision Systems, I collaborated with a surgeon and sterile processing technicians at the University of Maryland Medical Center to identify repetitive, time- consuming steps in the sterile processing procedure that could benefit from automation. Sterile processing technicians work to clean, sterilize, and pack surgical instruments so they are ready for upcoming procedures. We determined that automation of sorting and ordering of surgical implements after they had been cleaned and checked would provide significant benefit. I helped to integrate a six-degree-of-freedom robotic arm into the workflow by sourcing the arm, developing the software for detection and control, and designing custom end effectors. Staff saw potential to reduce human error, free time for higher-level quality control, and decrease post- surgical infection rates, while also improving job satisfaction and retention by relieving technicians of tedious tasks.

Object-Centric SLAM Research

At the Computational Engineering and Robotics Laboratory at CMU, I have developed a novel category-level object model for use in Semantic Simultaneous Localization and Mapping (SLAM) systems. Semantic SLAM goes beyond simply measuring an environment to perceiving it and mapping meaningful elements. Category-level object models are learned based on instances of objects within a class and provide compact, generalizable representations of these objects in a scene. My model leverages semantic keypoints and their relationships to represent object shape and improve the accuracy of measurements. This is the first object model used in SLAM that does not require solving for the object pose to inform measurements and estimate object shape. Objects can then be enrolled with fewer detected keypoints than previously. Additionally, articulated objects, such as backhoes, can maintain the same optimized shape while in many different configurations - something previously impossible and a barrier to using object shape for re-identification in dynamic environments. The implementation of this model has been a success jointly optimizing for object shape, location, and robot pose. Preliminary results show an improvement of 3.8% in localization RSME and 7.4 % in mapping RSME over not using an object model.

Farm Automation Robot

While interning at Reliance Roboics, a startup focused on developing low-cost agricultural robots to assist farmers with tasks such as monitoring fences, detecting predators, and tracking livestock, I focused on prototyping and software. I sourced and purchased sensors for the prototype, including RTK GPS, depth cameras, and IMUs, and implemented the magdwick sensor fusion algorithm to estimate orientation. In addition to hardware integration, I developed computer vision algorithms to follow farm staff and respond to hand gestures using deep learning models and OpenCV. Testing was a sucess, the robot was able to follow GPS waypoints with centimeter-level accuracy and respond to hand gestures in real-time.

Grounded ReID for SLAM

Data association in object-centric simultaneous localization and mapping frameworks allows for greater scene understanding, more accurate loop closure, and improved handling of dynamic objects. Currently most systems that utilize object based loop closure rely on mapping of class label based topographical object maps, however, in cases where objects may be dynamic, environments are cluttered, or in other cases of positional uncertainty these techniques commonly fail. On this project I am working to improve object-centric data association techniques by implementing grounded re-identification strategies based on deep learned embeddings to improve object tracking by associating observations of the same object across different viewpoints and time steps during training. I have sourced large (500k+ images) multi viewpoint datasets for the task and am currently testing several state of the art deep learning models including ConvNext and DEIT to extract robust feature embeddings for object re-identification. These feature embeddings are grounded with natural language embeddings formed from descriptions of the objects in the image and embedded with CLIP. Preliminary results show improvements in object tracking under large viewpoint changes when compared to feature embeddings alone.
